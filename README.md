## Task 1: Population Distribution Visualization

### Objective

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable.

### Dataset

World Bank Population Dataset

### Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

### Steps Performed

1. Loaded the population dataset.
2. Cleaned and processed the data.
3. Selected the latest population data.
4. Created a bar chart for the top 10 most populated countries.
5. Saved the visualization as an image.

### Outcome

Successfully visualized population distribution using a bar chart and gained hands-on experience with data visualization techniques.

### Skills Learned

* Data Cleaning
* Data Visualization
* Pandas
* Matplotlib
* Exploratory Data Analysis (EDA)
